### 三塘

|小区|备注|
|--|--|
|三塘桂园|东门离地铁站确实近，但是小区里面门洞钻来钻去不开阔。正门开在西北小道里|

1. 三塘这几个小区沿着香积寺路铺开，周边配套齐全，唯一缺点是没有大型商超
2. 离地铁站最近的是长兴公寓、菊园、桂园
